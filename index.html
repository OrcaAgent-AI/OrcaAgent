Framework Overview

OrcaAgent is an AI Agent development framework built on top of LangGraph. It deeply integrates with the rich ecosystem resources of LangChain/LangGraph and provides necessary encapsulation based on industry best practices.

Quick Start

1. Install CLI

pip install orcaagent-cli

2. Create New Project

orcaagent new my-project react

3. Configure Environment

# Copy environment template
cp .env.example .env

# Add your API keys
ANTHROPIC_API_KEY=your-key-here
TAVILY_API_KEY=your-key-here

4. Start Development

# Install dependencies
pip install -e .

# Start development server
orcaagent dev
